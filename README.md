# Data Analyst Salaries Dashboard

##  Project Overview

The Data Analyst Salaries Dashboard is an interactive Power BI dashboard designed to analyze salary trends of data professionals across different experience levels, company sizes, work years, employment types, remote work ratios, and job titles.

The dashboard helps users understand salary distributions and identify key factors that influence compensation in the data analytics industry.

---

## Tools & Technologies

- Power BI
- Microsoft Excel
- Data Visualization
- Data Cleaning
- DAX
- Power Query

## Dataset

The dataset contains salary information for data professionals, including:

- Work Year
- Experience Level
- Employment Type
- Company Size
- Remote Ratio
- Company Location
- Job Title
- Salary (USD)

##  Dashboard Features

- Interactive Filters (Slicers)
  - Work Year
  - Experience Level
  - Employment Type
  - Company Size

- KPI Cards
  - Total Employees
  - Average Salary
  - Total Salary
  - Highest Salary
  - Lowest Salary

- Visualizations
  - Average Salary by Experience Level
  - Employees by Remote Ratio
  - Average Salary by Work Year
  - Average Salary by Company Size
  - Average Salary by Company Location (Map)
  - Average Salary by Job Title
 
  ##  Key Insights

- Entry-level professionals have a competitive average salary compared to other experience levels.
- Fully remote jobs account for the highest percentage of employees.
- Salaries peaked in 2022 before showing a decline in later years.
- Medium-sized companies offer the highest average salaries.
- Machine Learning-related job roles receive the highest average salaries.
- Salary varies significantly across company locations and job titles.
